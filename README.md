# 🎬 Flix Website Clone

A **responsive front-end clone** of a popular streaming platform (inspired by Netflix), built using **HTML** and **CSS**.  
It mimics the layout, design, and responsiveness of a modern video streaming homepage.

### 🚀 Live Demo  
👉 [Visit the Website](https://abdullah-6203.github.io/flix-Website-Clone/)  

---

### 📱 Features

- ✅ Fully responsive (Mobile, Tablet, Desktop)
- 🎨 Clean UI with proper layout sections
- 🔘 Interactive buttons and hover effects
- 📺 Embedded videos & images for a real feel
- 💡 Designed using only **HTML & CSS** (no frameworks)

---

### 🛠️ Tech Stack

- HTML5
- CSS3
- Google Fonts
- SVG Icons
